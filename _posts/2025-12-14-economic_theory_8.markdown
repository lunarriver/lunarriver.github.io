本章深入探讨了税收对市场效率和经济福利的影响。

### 一、核心结论回顾

1. **税收在买者与卖者之间打入“楔子”**

- 无论向买者还是卖者征税，结果相同：买者支付价格上升，卖者实际所得下降。

- 市场均衡数量减少，市场规模缩小。

2. **税收归宿（Tax Incidence）由弹性决定**

- 税负更多地落在**缺乏弹性**的一方（即对价格变动反应较小的一方）。

- 例如：若需求缺乏弹性（如香烟），消费者承担大部分税负；若供给缺乏弹性（如短期住房），生产者承担更多。

### 二、关键概念：无谓损失（Deadweight Loss）

**定义**：税收导致的总剩余（消费者剩余 + 生产者剩余）减少中，**未被政府税收收入所抵消的部分**。

**成因**：税收扭曲激励，使部分原本互利的交易无法发生。

**图形表示**：供需曲线之间、征税后交易量减少区域形成的**三角形面积**（通常标记为 C + E）。

**公式表达**：  

```
征税前总福利 = 消费者剩余 + 生产者剩余

征税后总福利 = 新消费者剩余 + 新生产者剩余 + 税收收入

无谓损失 = 征税前总福利 − 征税后总福利
```

### 三、具体示例：清洁服务交易的消失

**背景**：

- 迈克（Mike）提供打扫服务，机会成本为 \$80，愿意以 \$100 出售。

- 梅（Mei）愿为打扫支付最高 \$120。

- 若无税：交易以 \$100 成交 → 消费者剩余 = \$20，生产者剩余 = \$20，总剩余 = \$40。

**征税 \$50 后**：

- 若梅付 \$120，迈克实得 \$70 < \$80（成本），不愿干；

- 若迈克要 \$80，梅需付 \$130 > \$120（估值），不愿买。

- **结果**：交易取消 → 双方各损失 \$20 剩余，政府税收 = \$0。

- **无谓损失 = \$40**（完全消失的互利交易价值）。

### 四、影响无谓损失大小的因素

1. **供给与需求的价格弹性**

- 弹性越大 → 对价格变动越敏感 → 交易量减少越多 → 无谓损失越大。

- 图形上：弹性大 → 供需曲线更平坦 → 无谓损失三角形面积更大。

**税率高低**

- 税率越高 → 价格楔子越大 → 市场萎缩越严重 → 无谓损失加速增长（按三角形面积 ∝ t² 增长）。

### 五、拉弗曲线（Laffer Curve）与政策启示

**核心观点**：税收收入并非随税率无限增加。

- 初期：提高税率 → 税收收入上升；

- 超过某临界点后：高税率严重抑制经济活动 → 税基萎缩 → 税收收入反而下降。

**政策含义**：

- 在高税率区间，**减税可能增加税收收入**（里根经济学的理论基础）。

- 税制设计应权衡**效率（最小化无谓损失）与公平（收入再分配）**。

### 六、总结要点

| 概念 | 内容 |
|------|------|
| **税收楔子** | 买者支付价与卖者实得价之差 = 税额 |
| **税收归宿** | 由供需弹性决定，税负落在缺乏弹性一方 |
| **无谓损失** | 因税收阻止互利交易而造成的社会总剩余净损失 |
| **弹性作用** | 弹性越大，无谓损失越大 |
| **拉弗曲线** | 税收收入先增后减，存在最优税率 |

通过本章分析可见：**税收不仅是财富转移工具，更是改变行为、扭曲效率的政策手段**。理解其“代价”，有助于设计更高效、更公平的税制。

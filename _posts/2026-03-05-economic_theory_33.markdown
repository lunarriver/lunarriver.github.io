# 一、本章核心模型

**总需求 AD + 总供给 AS = 宏观经济均衡**

决定：

- **物价总水平 P**
- **总产出（真实 GDP）Y**

# 二、总需求 AD（向下倾斜）

AD=C+I+G+NX

## 为什么向下倾斜？3 个原因（必考）

1. **财富效应**
   
   物价↓ → 货币更值钱 → 消费 C↑ → AD↑
   
   例：物价跌，你手里钱更值钱，更敢消费。

2. **利率效应**
   
   物价↓ → 货币需求↓ → 利率↓ → 投资 I↑ → AD↑
   
   例：物价低→利率低→企业多借钱建厂。

3. **汇率效应**
   
   物价↓ → 利率↓ → 本币贬值 → NX↑ → AD↑
   
   例：国内物价低，出口更好卖。

## 使 AD 移动的因素（右 = 扩张，左 = 收缩）

- 消费信心↑、减税 → C↑ → **AD 右移**
- 企业乐观、利率↓ → I↑ → **AD 右移**
- 政府支出↑ → G↑ → **AD 右移**
- 外国繁荣、本币贬值 → NX↑ → **AD 右移**

# 三、总供给 AS（三种情况）

## 1. 长期总供给 LRAS（垂直）

- 长期：产出由**技术、资源、制度**决定，**与物价无关**
- 产出 = **自然产出水平（潜在 GDP）**

## 2. 短期总供给 SRAS（向右上倾斜）

为什么短期向上？3 个理论（记名字就行）

1. **黏性工资理论**：工资慢，物价涨企业愿意多生产
2. **黏性价格理论**：部分价格不灵活
3. **错觉理论**：企业误以为是自己商品涨价

## 使 SRAS 移动的因素

- 工资↓、原料价↓、技术进步 → **SRAS 右移**
- 工资↑、油价↑、预期物价↑ → **SRAS 左移**

# 四、三大宏观现象（本章灵魂）

## 1. 长期均衡

- AD 与 LRAS、SRAS 交于一点
- 产出 = 潜在 GDP
- 充分就业

## 2. 经济衰退（总需求减少）

- AD **左移**
- 产出 Y↓，物价 P↓
- 失业↑

例子：

疫情恐慌 → 消费投资下降 → AD 左移 → 衰退。

**长期自我修复**：

预期物价↓ → 工资↓ → SRAS 右移 → 回到潜在产出。

## 3. 滞胀（最讨厌：失业 + 通胀）

原因：**供给冲击**（SRAS 左移）

- 油价暴涨、工资大涨
- 产出 Y↓，物价 P↑
- 滞胀 = **停滞 + 通胀**

例子：

70 年代石油危机 → 成本暴涨 → 滞胀。

# 五、本章最关键结论

1. **长期：货币中性，只影响物价，不影响产出**
2. **短期：物价黏性，总需求影响产出与就业**
3. **衰退多来自总需求不足**
4. **滞胀一定来自供给侧冲击**

# 总结

- **总需求 AD 向下倾斜：财富、利率、汇率效应**
- **长期总供给垂直，短期向上倾斜**
- **AD 动 → 波动；AS 动 → 滞胀**
- 长期经济会**自动回到潜在产出**。

# 一、本章核心一句话

金融体系的作用：**把储蓄者的钱，融通给投资者（企业 / 政府）**，让资源从**储蓄**流向**投资**。

# 二、关键概念（必背）

## 1. 金融体系两类机构

**金融市场**：资金直接从储蓄者流向借款者
   
- 股票市场

- 债券市场

**金融中介**：资金间接流动
   
- 银行

- 共同基金

## 2. 债券 vs 股票

**债券**：债务凭证，到期还本付息，**无所有权**
  
例：腾讯发 5 年期债券，你买 = 借钱给腾讯。

**股票**：所有权凭证，**分享利润、共担风险**
  
例：买阿里巴巴股票 = 你是股东。

## 3. 国民收入恒等式（超级重点）

Y=C+I+G+NX

**封闭经济**（不考虑国际贸易）：

Y=C+I+G

变形得到：

国民储蓄 = 投资

这是本章**最核心结论**：

**对整个国家而言，储蓄 = 投资**。

# 四、国民储蓄 S 的拆分

S=(Y−T−C)+(T−G)

**私人储蓄**：Y−T−C
  
家庭收入 - 税 - 消费，剩下存起来的钱。

**公共储蓄**：T−G
  
政府收入 - 政府支出。

1. T>G → **预算盈余**（公共储蓄为正）

2. T<G → **预算赤字**（公共储蓄为负）

# 五、可贷资金市场（本章模型）

把金融体系简化成**一个市场**：

- 供给方：**储蓄者**

- 需求方：**想投资的人 / 企业**

- 价格：**真实利率 r**

### 1. 供给曲线

储蓄越多 → 可贷资金供给越多 → 向右上方倾斜。

### 2. 需求曲线

利率越低 → 企业越愿意借钱投资 → 向右下方倾斜。

### 3. 均衡

利率调整到 **储蓄 = 投资**。

# 六、三个经典政策分析（必考）

## 1. 储蓄激励（如减税鼓励储蓄）

- 供给曲线右移

- 均衡利率**下降**

- 投资**增加**

例：政府对利息收入免税 → 大家更愿意存钱 → 可贷资金变多 → 利率下降 → 企业多投资。

## 2. 投资激励（如投资税抵免）

- 需求曲线右移

- 均衡利率**上升**

- 储蓄与投资**都增加**

例：政府给买设备的企业减税 → 企业更想投资 → 借钱变多 → 利率上升。

## 3. 政府预算赤字

- 公共储蓄减少 → 总储蓄减少

- 供给曲线**左移**

- 利率**上升**

- 私人投资**被挤出**（挤出效应）

例：政府大量借钱发债 → 市场钱变少 → 利率涨 → 企业少借钱建厂。

# 七、本章最容易混淆的点

- **日常说的 “投资”**（买股票、买基金）
  
在曼昆里叫：**储蓄**。

- **曼昆说的 “投资 I”**
  
是**实物投资**：建厂房、买机器、建住房。

**一句话区分：**

- 你买股票 / 债券 → **储蓄**

- 企业买设备 / 建楼 → **投资 I**

# 八、极简总结

1. 金融体系连接**储蓄**与**投资**。

2. 封闭经济：**国民储蓄 = 投资**。

3. 可贷资金市场：
   
- 供给 = 储蓄

- 需求 = 投资

- 价格 = 真实利率

4. 赤字会**推高利率、挤出投资**。

5. 鼓励储蓄 / 投资的政策会提高长期经济增长。

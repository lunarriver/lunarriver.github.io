# 一、本章核心结论

- **货币政策**与**财政政策**都会影响**总需求 AD**，从而影响短期产出、就业、物价。
- 重点：**乘数效应**、**挤出效应**、**利率效应**。

# 二、货币政策（央行控制）

## 1. 传导机制

- 货币供给↑ → 利率↓ → 投资↑ → **总需求 AD 右移** → 产出 Y↑、就业↑
- 货币供给↓ → 利率↑ → 投资↓ → **总需求 AD 左移** → 产出 Y↓、通胀↓

**例子：**

央行降息 / 放水 → 企业借钱建厂、居民买房买车 → AD 上升 → 经济回暖。

# 三、财政政策（政府控制）

两种工具：

1. **政府购买 G**（直接）
2. **税收 T**（间接）

---

## 1. 乘数效应

### （1）政府购买乘数

政府购买乘数

- MPC：边际消费倾向（每多赚 1 元，花掉多少）

**例子：**

MPC = 0.8 → 乘数 = 5

政府多花 100 亿 → 总需求最终增加 **500 亿**。

### （2）税收乘数

税收乘数

- 减税 → 可支配收入↑ → C↑ → AD↑

**例子：**

MPC = 0.8 → 税收乘数 = -4

减税 100 亿 → AD 增加 **400 亿**。

---

## 2. 挤出效应

- 政府支出↑ → 总需求↑ → 货币需求↑ → **利率↑** → 投资↓ → AD 被 “挤掉” 一部分。

**结论：**

**乘数效应扩大 AD，挤出效应缩小 AD，最终效果看谁更强。**

# 四、两种政策对比

## 1. 货币政策

- 工具：货币供给、利率
- 优点：快、灵活
- 缺点：衰退时**流动性陷阱**（利率很低，再降也没用）

## 2. 财政政策

- 工具：政府支出、税收
- 优点：衰退时**直接拉动 AD**，效果强
- 缺点：慢、有挤出效应

# 五、两大政策规则

1. **自动稳定器**
   
   - 不需要政府主动做，经济差时自动刺激：
     
     失业救济、累进税
   
   - 衰退 → 税收自动↓、支出↑ → AD↑

2. **政策时滞**
   
   - 货币政策：执行快，效果慢
   - 财政政策：立法慢，执行慢

# 六、总结

**货币政策通过利率影响投资；

财政政策通过乘数扩大总需求，但会被挤出效应抵消一部分；

两者都是为了在短期稳定总需求、稳定经济。**

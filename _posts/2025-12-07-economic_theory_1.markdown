#### 原理1：人们面临权衡取舍

```
“天下没有免费的午餐。”
```

**基本内涵**

资源是稀缺的，为了获得某种东西，必须放弃另一种东西。所有决策都涉及取舍，例如在“效率”与“平等”之间权衡。

**重要概念及内涵**

- 稀缺性（Scarcity）：资源有限而欲望无限，是经济学存在的前提。

- 权衡（Trade-off）：为得到A而放弃B的选择关系。

- 效率（Efficiency）：社会从资源中获得最大产出（“把蛋糕做大”）。

- 平等（Equity）：经济成果在成员间公平分配（“如何分蛋糕”）。

**应用实例**

学生选择用一小时复习经济学而非打游戏，牺牲娱乐换取知识。

政府增加军费开支，可能减少教育或医疗投入。

#### 原理2：某种东西的成本是为了得到它所放弃的东西

```
成本 = 机会成本
```

**基本内涵**

真正的成本不是账面支出，而是机会成本——你为某项选择而放弃的最佳替代选项的价值。

**重要概念及内涵**

- 机会成本（Opportunity Cost）：选择某一选项时所放弃的最高价值替代方案。

- 显性成本（Explicit Cost）：需实际支付货币的成本（如工资、租金）。

- 隐性成本（Implicit Cost）：未发生现金支出但存在机会成本的代价（如自用房屋放弃的租金）。

**应用实例**

上大学四年的机会成本包括学费（显性）+ 若工作可赚取的工资（隐性）。

企业主用自己的资金创业，放弃了将这笔钱存银行获得的利息。

#### 原理3：理性人考虑边际量

```
决策发生在“边缘”上
```

**基本内涵**

理性人通过比较边际收益（MB）与边际成本（MC）来做决策，最优选择发生在 MB = MC 时。

**重要概念及内涵**

- 边际收益（Marginal Benefit, MB）：多消费或生产一单位带来的额外收益。

- 边际成本（Marginal Cost, MC）：多生产一单位引起的额外成本。

- 最优决策规则：当 MB > MC 时应继续；MB < MC 时应停止；MB = MC 时净收益最大。

```
MB = MC，表示一个临界点，此时总利润最大，在此之前，总利润有上升空间，在此之后，总利润则会下降。

MB = MC 是最后一笔不亏不赚但能保住最大总利润的决策。
```

**应用实例**

航空公司以低于平均票价的价格出售最后一张机票，只要票价 > 边际成本（如餐食、燃油微增），就值得卖。

吃到第三块披萨已饱，第四块的满足感（边际效用）很低，不值得再吃。

#### 原理4：人们会对激励做出反应

```
行为随成本与收益变化而调整
```

**基本内涵**

当政策、价格或制度改变行为的“激励”时，人们会相应调整其选择。好的政策需预见行为反应。

重要概念及内涵

- 激励（Incentive）：影响行为的奖励或惩罚机制（如价格、税收、补贴）。

- 行为反应（Behavioral Response）：个体对激励变化的适应性调整。

- 意外后果（Unintended Consequences）：政策忽视激励效应可能导致反效果。

**应用实例**

汽车安全带法规降低司机死亡率，但也可能鼓励更冒险驾驶（道德风险），增加行人事故。

对高收入者大幅加税，可能削弱其工作或投资积极性。

#### 原理5：贸易可以使每个人的状况都变得更好

```
自愿交换创造双赢
```

**基本内涵**

通过专业化和基于比较优势的贸易，各方都能以更低机会成本获得更多商品，实现互利。

**重要概念及内涵**

- 比较优势（Comparative Advantage）：以较低机会成本生产某种商品的能力（≠绝对优势）。

- 专业化（Specialization）：集中生产自己机会成本最低的产品。

- 互惠贸易（Mutually Beneficial Trade）：双方自愿交换，均认为自身获益。

**应用实例**

中国擅长制造服装，美国擅长设计芯片，两国贸易后都能以更低价格获得更多商品。

家庭中丈夫修车快、妻子做饭快，分工合作比各自都做更高效。

#### 原理6：市场通常是组织经济活动的好方法

```
“看不见的手”引导资源有效配置
```

**基本内涵**

在自由市场中，价格机制协调千百万人的分散决策，高效配置资源，无需中央计划。

**重要概念及内涵**

- “看不见的手”（Invisible Hand）（亚当·斯密）：个体追求私利时，无意中促进社会福利。

- 价格信号（Price Signals）：价格反映稀缺性和偏好，引导生产与消费。

- 分散决策（Decentralized Decision-Making）：无数买家卖家自主行动，形成协调结果。

**应用实例**

苹果热销 → 价格上涨 → 果农多种苹果、商家多进货 → 供给增加 → 价格回落，实现均衡。

计划经济（如苏联）因缺乏价格信号，常导致短缺或浪费。

#### 原理7：政府有时可以改善市场结果

```
纠正市场失灵
```

**基本内涵**

当市场因外部性、垄断、公共品等原因无法有效运行时，政府干预可提升效率或公平。

**重要概念及内涵**

- 市场失灵（Market Failure）：市场无法有效配置资源的情形。

- 外部性（Externality）：行为对旁观者造成未补偿的影响。

- 负外部性（如污染）→ 社会成本 > 私人成本 → 过度生产

- 正外部性（如教育）→ 社会收益 > 私人收益 → 生产不足

- 公共物品（Public Goods）：非排他性 + 非竞争性（如国防、路灯），私人市场供给不足。

- 市场势力（Market Power）：单个主体能影响价格（如垄断），导致产量低于社会最优。

**应用实例**

工厂排污 → 政府征收碳税或设定排放上限。

国防由政府提供，因私人无法排除“搭便车者”。

#### 原理8：一国的生活水平取决于它生产物品与服务的能力

```
生产率决定长期繁荣
```

**基本内涵**

人均收入长期由劳动生产率决定，而生产率取决于技术、资本、制度和人力资本。

**重要概念及内涵**

- 生产率（Productivity）：每单位劳动投入所生产的产出量。

- 人均GDP（Per Capita GDP）：衡量平均生活水平的核心指标。

- 人力资本（Human Capital）：劳动者通过教育、培训积累的知识与技能。

- 物质资本（Physical Capital）：机器、设备、基础设施等生产工具。

- 技术进步（Technological Progress）：新知识或工艺使同样投入产出更多，是增长的根本动力。

**应用实例**

德国工人使用先进自动化设备，每小时产出高 → 工资和生活水平高。

撒哈拉以南非洲国家因教育落后、基础设施差，生产率低 → 收入水平低。

#### 原理9：当政府发行了过多货币时，物价上涨

```
通货膨胀是一种货币现象
```

**基本内涵**

若货币供给增长快于实际产出增长，将导致物价总水平持续上升，即通货膨胀。

**重要概念及内涵**

- 通货膨胀（Inflation）：物价普遍、持续上涨，货币购买力下降。

- 货币供给（Money Supply）：经济中流通的货币总量，由中央银行控制。

- 货币数量论（Quantity Theory of Money）：公式 MV = PY，若V（流通速度）和Y（产出）稳定，则M↑ → P↑。

**应用实例**

1920年代德国恶性通胀：政府印钞还债 → 马克急剧贬值 → 买面包需推车装钱。

近年土耳其、阿根廷因财政赤字货币化，通胀率飙升至50%以上。

#### 原理10：社会面临通货膨胀与失业之间的短期权衡取舍

```
菲利普斯曲线揭示短期矛盾
```

**基本内涵**

短期内，刺激总需求可降低失业，但会推高通胀；反之亦然。这种权衡由菲利普斯曲线描述。

**重要概念及内涵**

- 菲利普斯曲线（Phillips Curve）：短期中通胀率与失业率呈反向关系。

- 总需求（Aggregate Demand, AD）：整个经济对商品与服务的总支出。扩张性政策（如降息）提升AD。

- 经济周期（Business Cycle）：经济围绕长期趋势的波动（繁荣、衰退等），政策常试图平抑波动。

**应用实例**

2008年金融危机后，美联储实施量化宽松 → 失业率从10%降至4%，但2021–2022年通胀升至9%。

政府为保就业扩大基建投资，短期内企业招人、需求上升，但可能引发物价压力。

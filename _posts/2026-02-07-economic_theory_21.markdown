曼昆《经济学原理（微观经济学分册）》第21章“消费者选择理论”是理解个体如何在有限资源下做出最优消费决策的核心章节。以下是该章的**核心概念总结**与**具体示例说明**：

### 一、两大基础工具

#### 1. **预算约束（Budget Constraint）**

**定义**：表示在给定收入和价格下，消费者能够负担得起的所有商品组合。

**数学表达式**：

$$
P_X \cdot X + P_Y \cdot Y = M
$$

其中 $P_X, P_Y$ 是商品X和Y的价格，$M$ 是收入。

**图形特征**：

- 预算线是一条**向下倾斜的直线**；

- **斜率 = $-\frac{P_X}{P_Y}$**，其绝对值代表**相对价格**（即市场要求用多少单位Y换1单位X）；

- 收入增加 → 预算线**平行外移**；某商品价格下降 → 预算线绕轴**旋转外扩**。

**示例**：

- 假设你每月有 \$60，可乐（X）\$2/瓶，披萨（Y）\$3/个。

- 预算方程：$2X + 3Y = 60$

- 若全买可乐：X=30；全买披萨：Y=20。 

- 预算线斜率 = $-2/3$，意味着每多买1个披萨，必须少买1.5瓶可乐。

#### 2. **无差异曲线（Indifference Curve）**

**定义**：表示给消费者带来**相同满足程度（效用）** 的所有商品组合。

**四大特征**：

1. **越高越好**：离原点越远的曲线代表更高效用；

2. **向下倾斜**：要维持效用不变，多一种商品必须少另一种；

3. **不相交**：否则违反“偏好一致性”；

4. **凸向原点**：反映**边际替代率递减**（MRS↓）。

**边际替代率（MRS）**：

- 定义：消费者愿意用Y换取X的比率，保持效用不变；

- 公式：$\text{MRS} = \left| \frac{\Delta Y}{\Delta X} \right| = \frac{MU_X}{MU_Y}$；

- 凸性 ⇒ MRS 随X增加而**递减**（你拥有的X越多，越不愿用Y换它）。

**示例**：  

- 在A点（X=2, Y=10），你愿意用2单位Y换1单位X（MRS=2）；  

- 在B点（X=8, Y=3），你只愿用0.5单位Y换1单位X（MRS=0.5）——因为X已很多。

### 二、消费者最优选择（Consumer Optimum）

**条件**：**最高可达的无差异曲线**与**预算线相切**。

**切点满足**：

$$
\text{MRS} = \frac{P_X}{P_Y} \quad \text{或等价地} \quad \frac{MU_X}{P_X} = \frac{MU_Y}{P_Y}
$$

含义：**每花1元在X或Y上带来的边际效用相等**，即“钱花得最值”。

**示例**：  

- 可乐 MU=10，价格\$2 → 每元边际效用 = 5；  

- 披萨 MU=15，价格\$3 → 每元边际效用 = 5。  

- 此时达到最优。若可乐 MU 升至 12，则每元效用=6 > 披萨的5，应多买可乐直到均衡恢复。

**特殊情况**：**角点解（Corner Solution）**

- 当商品为**完全替代品**（如1元硬币 vs 2枚5角硬币）→ 无差异曲线为直线 → 最优选极端点（全买便宜的那个）；

- 当商品为**完全互补品**（如左鞋与右鞋）→ 无差异曲线为直角 → 最优在“拐点”。

### 三、收入与价格变动的影响

#### 1. **收入变化**

收入↑ → 预算线**平行外移** → 消费者移向更高无差异曲线。

- 若某商品消费量↑ → **正常品（Normal Good）**；

- 若消费量↓ → **低档品（Inferior Good）**（如方便面、公交卡）。

**示例**：收入增加后，你多吃牛排（正常品），少吃泡面（低档品）。

#### 2. **价格变化（以X降价为例）**

预算线绕Y轴**向外旋转**（X最大购买量增加）；

总效应 = **替代效应 + 收入效应**：

- **替代效应**：X相对变便宜 → 多买X（总是负向：价格↓→需求↑）；

- **收入效应**：实际购买力上升 → 对正常品多买，对低档品少买。

**特殊案例：吉芬商品（Giffen Good）**  

极端低档品，收入效应 > 替代效应 → 价格↑反而需求↑（如饥荒时的土豆）。**极其罕见**。

### 四、理论意义

- 解释了**需求曲线为何向下倾斜**（一般情况下）；

- 揭示了消费者行为背后的**理性最大化逻辑**；

- 为分析税收、补贴、工资变化等政策提供微观基础。

### 总结

```
Y（披萨）
│
│    I₃（高满足）
│     \
│      \ I₂ ← 最优切点（MRS = P_X/P_Y）
│       \
│--------\-------- 预算线（2X + 3Y = 60）
│         \
│          \ I₁（低满足）
└─────────────── X（可乐）
```

消费者在预算限制下，选择**切点**实现效用最大化。

本章将**约束（预算）** 与**偏好（无差异曲线）** 结合，构建了现代微观经济学分析个体选择的基石框架。

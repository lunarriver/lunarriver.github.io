本章系统地分析了垄断市场的特征、形成原因、企业行为、福利影响以及政府可能采取的公共政策。

### 一、垄断的基本特征

**1. 垄断的定义**

垄断是指市场上只有一个卖者，且其产品没有相近替代品。该企业是**价格制定者**（price maker），而非竞争市场中的价格接受者。

**2. 垄断市场的三大条件**：

- 单一卖者（sole seller）

- 无相近替代品（no close substitutes）

- 存在进入壁垒（barriers to entry）

**3. 需求曲线特征**  

垄断企业面临的是**整个市场需求曲线**，向右下方倾斜。这意味着：要多卖一单位产品，必须降低价格。

### 二、垄断的成因

曼昆指出，垄断通常源于以下三类**进入壁垒**：

| 成因 | 说明 | 示例 |
|------|------|------|
| **关键资源控制** | 企业独占生产所需的关键资源 | De Beers 曾控制全球大部分钻石供应 |
| **政府授权** | 政府授予专利、版权或特许经营权 | 药企对新药拥有20年专利（如辉瑞的伟哥） |
| **自然垄断** | 规模经济导致单个厂商成本最低 | 自来水、电网、铁路等基础设施行业 |

**案例**：当一种新药专利有效时，制药公司是唯一供应商，可定高价；专利到期后，仿制药（无品牌药）涌入，价格大幅下降——这验证了从垄断到竞争的转变。


### 三、垄断企业的决策：产量与定价

**核心规则**：垄断企业通过 **边际收益 = 边际成本**（MR = MC）来决定最优产量，再根据需求曲线确定价格。

**关键关系**：

- **MR < P**：因为降价不仅影响新增销量，也降低原有销量的收入。

- **P > MC**：垄断价格高于边际成本，导致资源配置无效率。

**利润公式**：  

利润 = (P - ATC) * Q

**举例**（脑残片模型）：若药价从90元降至89元，销量从10人增至11人：

- 新增收入：89元

- 原有10人每少收1元 → 损失10元

- **边际收益 = 89 - 10 = 79元 < 89元（价格）**

### 四、垄断的社会成本：无谓损失（Deadweight Loss）

**完全竞争市场**：P = MC，总剩余最大，无效率损失。

**垄断市场**：

- 产量低于社会最优水平（Q_m < Q_c）

- 价格高于边际成本（P_m > MC）

- 产生**无谓损失**（DWL）：即本可实现但未实现的社会福利。

图形上，DWL 是需求曲线、边际成本曲线与垄断产量之间的三角形区域。

### 五、价格歧视（Price Discrimination）

**定义**：对不同消费者收取不同价格，以攫取更多消费者剩余。

**三种类型**：

1. **一级价格歧视**（完全歧视）：按每个消费者最高支付意愿定价 → 消费者剩余为零，无 DWL（效率高但不公平）。

2. **二级价格歧视**：按购买数量定价（如“买二送一”）。

3. **三级价格歧视**：按群体定价（如学生票、老人票）。

**现实例子**：航空公司对商务旅客（高支付意愿）收高价，对提前购票的休闲旅客收低价。

### 六、针对垄断的公共政策

| 政策 | 说明 | 优缺点 |
|------|------|--------|
| **反托拉斯法**（反垄断法） | 禁止合谋、拆分垄断企业 | 如1911年标准石油被拆分为埃克森、美孚等；但可能抑制创新 |
| **价格管制** | 政府设定最高价 | 如公交票价长期不变；但若价格=MC，自然垄断企业可能亏损 |
| **公有制** | 政府直接经营 | 如法国电力公司；但可能效率低下、滋生腐败 |
| **不作为** | 允许垄断存在 | 依赖潜在竞争（如打火机替代火柴）；但短期消费者受损 |

**建桥案例**：修桥成本200万，最大收益160万 → 私人不会建（亏损），但社会价值高 → **政府应建桥并免费开放**（因 MC=0），体现正外部性。

### 七、核心结论

1. 垄断通过限制产量、提高价格获取利润，但造成**社会福利净损失**。

2. 垄断不一定总是坏的：**专利制度**激励创新，**自然垄断**避免重复建设。

3. 政策需权衡效率与公平、短期利益与长期激励。

4. **价格歧视**可能减少无谓损失，但引发公平问题。

#### 总结

```
完全竞争： P = MC，Q = Q_c，总剩余 = A+B+C
垄断：     P = P_m > MC，Q = Q_m < Q_c
           消费者剩余 = A
           生产者剩余 = B
           无谓损失 = C
```

其中 A、B、C 为图形中的典型区域。

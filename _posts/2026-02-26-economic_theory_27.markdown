# 一、本章核心主题

讲**个人 / 企业如何在**不确定、跨时间**的条件下做金融决策**。

三大工具：

1. **现值**（钱的时间价值）

2. **风险管理**（风险厌恶、保险、多元化）

3. **资产估值**（基本面分析、有效市场假说）

# 二、核心 1：现值与未来值（钱的时间价值）

## 1. 现值 PV = 未来钱在今天值多少

- 未来值 FV = 现值 × (1 + r)^n

- 现值 PV = 未来值 / (1 + r)^n

**r = 利率，n = 年数**

### 示例

利率 5%，10 年后的 10000 元，今天值多少？

PV = 10000 / (1.05)¹⁰ ≈ **6139 元**

### 两个重要结论

1. **利率越高 → 现值越低**

2. **时间越久 → 现值越低**

## 2. 70 法则（快速估算翻倍时间）

- 价值翻倍时间 ≈ **70 / 增长率 (%)**

### 示例

利率 2%：

70/2 = **35 年翻倍**

利率 7%：

70/7 = **10 年翻倍**

# 三、核心 2：风险管理

## 1. 风险厌恶

- 同样概率，**失去 100 元的痛苦 > 得到 100 元的快乐**

- 人们愿意花钱**减少不确定性**

## 2. 保险的作用

- 不是消除风险，而是**分摊风险、分散风险**

- 面临两个问题：
  
1. **逆向选择**：高风险的人更想买保险

2. **道德风险**：买了保险后行为更冒险

## 3. 多元化：不要把鸡蛋放一个篮子

- 能消除**特有风险**（单个公司 / 行业风险）

- 不能消除**市场风险**（整个经济下跌）

### 示例

- 只买茅台股票：风险大（特有风险）

- 买全市场指数基金：只剩市场风险，更安全

# 四、核心 3：资产价格如何决定

## 1. 基本面分析

看一家公司**真实价值**：

- 未来盈利、分红、资产、竞争力

## 2. 有效市场假说 EMH（本章重点）

- 股价**已经反映所有公开信息**

- 市场是**信息有效**的

- 价格波动是**新消息**导致的，不可预测

- 结论：**很难长期跑赢市场，指数基金更靠谱**

## 3. 市场非理性

- 很多人认为市场会有**泡沫、恐慌、羊群效应**

- 股价有时偏离基本面

# 五、总结

1. 今天的钱比未来的钱值钱，因为有**利息**。

2. 现值越低，说明**利率越高、时间越久**。

3. **70 法则**用来估算价值翻倍时间。

4. 多元化能消除**企业特有风险**，不能消除**市场风险**。

5. 有效市场假说认为：**股价反映所有信息，难以战胜市场**。

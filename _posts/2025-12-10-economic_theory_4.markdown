本章解释市场经济如何通过价格机制配置稀缺资源。

### 一、市场与竞争

**市场（Market）**：由某种物品或服务的买者与卖者组成的群体。 例如，橙汁市场包括所有想买橙汁的消费者和所有生产/销售橙汁的企业。

**竞争市场（Competitive Market）**：有许多买者和卖者，每个人对市场价格影响微乎其微。

竞争市场的特征（完全竞争）：

- 商品同质（如小麦、自来水）

- 买卖者数量众多

- 自由进出市场

- 信息完全

**市场类型对比**：

| 市场类型       | 卖家数量 | 产品差异 | 价格控制力 |
|----------------|----------|----------|------------|
| 完全竞争       | 极多     | 无差异   | 无         |
| 垄断竞争       | 多       | 有差异   | 小         |
| 寡头           | 少（2–5）| 可同可异 | 中         |
| 完全垄断       | 1        | 唯一     | 强         |

例如，蜜雪冰城 vs 喜茶 → 垄断竞争；中国移动/联通/电信 → 寡头；小镇唯一自来水公司 → 完全垄断。

### 二、需求（Demand）

**需求量（Quantity Demanded）**：消费者在某一价格下 **愿意且能够购买** 的数量。注意，“愿意+能够”缺一不可。

**需求定理（Law of Demand）**：在其他条件不变时，商品价格上升 → 需求量下降（反向关系）。

需求曲线向右下方倾斜。

**市场需求 = 所有个体需求之和**（水平加总）。

**需求 vs 需求量**：

- **需求量变动**：价格变化 → 沿着需求曲线移动。

- **需求变动**：非价格因素变化 → 整条曲线左右平移。

**导致需求曲线移动的因素（外生变量）**：

（1）**收入**：

- 正常品（Normal Good）：收入↑ → 需求↑（如火锅）
- 低档品（Inferior Good）：收入↑ → 需求↓（如泡面）

（2）**相关商品价格**：

- 替代品（Substitutes）：A价格↑ → B需求↑（如冰淇淋 vs 冷冻酸奶）

- 互补品（Complements）：A价格↑ → B需求↓（如电脑 ↑ → 软件 ↓）

（3）**偏好（Tastes）**：健康风潮 → 健身器材需求↑

（4）**预期（Expectations）**：预期汽油涨价 → 今天多加油

（5）**买者数量**：人口增长 → 市场需求↑

例如，中东战争 → 汽油价格飙升 → 人们减少购买耗油的二手凯迪拉克（互补关系），其需求曲线左移，价格下跌。

### 三、供给（Supply）

**供给量（Quantity Supplied）**：生产者在某一价格下 **愿意且能够提供** 的数量。

**供给定理（Law of Supply）**：在其他条件不变时，商品价格上升 → 供给量上升（正向关系）。

供给曲线向右上方倾斜。

**市场供给 = 所有个体供给之和**。

**供给 vs 供给量**：

- **供给量变动**：价格变化 → 沿供给曲线移动。

- **供给变动**：非价格因素 → 供给曲线平移。

**导致供给曲线移动的因素**：

（1）**投入品价格**：橙子冻坏（原料成本↑）→ 橙汁供给↓（左移）

（2）**技术进步**：自动化 → 成本↓ → 供给↑

（3）**预期**：预期未来手机涨价 → 厂商囤货，当前供给↓

（4）**卖者数量**：新奶茶店开业 → 市场供给↑

例如，佛罗里达寒流 → 橙子减产 → 橙汁供给减少 → 价格上涨。

### 四、市场均衡（Equilibrium）

**均衡价格（Equilibrium Price）**：使供给量 = 需求量的价格。

**均衡数量（Equilibrium Quantity）**：此时交易的数量。

**供求定理**：价格会自动调整，消除过剩或短缺，趋向均衡。

- 价格 > 均衡价 → **过剩（Surplus）** → 价格下降

- 价格 < 均衡价 → **短缺（Shortage）** → 价格上升

例如，夏天加勒比酒店空房多（供给↑ 或 需求↓）→ 价格下降促销。

### 五、分析市场变动的三步法

（1）**判断**：事件影响供给？需求？还是两者？

（2）**方向**：曲线向左还是向右移动？

（3）**比较**：新旧均衡 → 价格和数量如何变化？

例如，电动汽车补贴政策  

- → 降低购车成本 → 需求↑（需求曲线右移）  

- → 均衡价格↑，均衡数量↑

### 六、价格作为资源配置信号

- 在市场经济中，**价格是“看不见的手”**，引导资源流向最需要的地方。

- 高价激励生产（供给↑），抑制消费（需求↓）；低价反之。

- 最终实现稀缺资源的有效配置。

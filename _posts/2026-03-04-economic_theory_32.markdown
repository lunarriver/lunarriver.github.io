本章核心模型：

可贷资金市场 + 外汇市场 = 开放经济一般均衡

两个市场同时平衡，决定：**真实利率 r** 和 **真实汇率 ε**。

## 一、两个市场、两条曲线

### 1. 可贷资金市场（决定：真实利率 r）

- 供给：国民储蓄 S
- 需求：国内投资 I
- 价格：**真实利率 r**

**恒等式：**

供给 = 需求

S = I + NCO

- 利率上升 → 储蓄增加、投资减少、**资本净流出 NCO 减少**
- 利率下降 → 储蓄减少、投资增加、**资本净流出 NCO 增加**

### 2. 外汇市场（决定：真实汇率 ε）

- 供给：**资本净流出 NCO**（换外币，供给本币）
- 需求：**净出口 NX**（卖商品赚外币，需求本币）
- 价格：**真实汇率 ε**

**恒等式（本章灵魂）：**

NCO = NX

- 汇率升值 → 出口变贵、进口便宜 → **NX ↓**
- 汇率贬值 → 出口便宜、进口变贵 → **NX ↑**

## 二、两个市场的**唯一连接点**

**资本净流出 NCO**

- 在**可贷资金市场**：是需求的一部分

- 在**外汇市场**：是**供给曲线**

- **真实利率 r 决定 NCO**：
  
  - r 上升 → 国内资产更吸引人 → **NCO 下降**
  - r 下降 → 国内资产不吸引人 → **NCO 上升**

## 三、三个经典政策实验（必考！）

### 1. 政府预算赤字（减少国民储蓄）

1. 储蓄 S ↓ → 可贷资金供给左移
2. **真实利率 r ↑**
3. 利率高 → **NCO ↓**
4. 外汇市场本币供给 ↓ → **真实汇率 ε ↑**（升值）
5. 升值 → **NX ↓**（贸易逆差扩大）

**结论：**

**预算赤字 → 利率上升 → 本币升值 → 净出口减少**

（这就是**双赤字**：财政赤字 + 贸易赤字）

### 2. 进口配额（贸易保护）

1. 限制进口 → **NX 需求右移**
2. **真实汇率 ε ↑**（升值）
3. 升值 → 出口减少、进口变相增加
4. **NX 最终不变！**
5. 汇率升值完全抵消保护效果

**结论：**

**贸易保护不能改变贸易余额，只会让本币升值，出口跟着减少。**

### 3. 资本外逃（政治风险）

1. 人们恐慌 → **想把资产转移国外 → NCO 大幅右移**
2. 可贷资金需求 ↑ → **真实利率 r ↑**
3. 外汇市场本币供给 ↑ → **真实汇率 ε ↓**（大幅贬值）
4. 贬值 → **NX ↑**（出口暴增）

**例子：**

某国政局动荡 → 资金跑路 → 利率飙升、货币暴跌、出口变很强。

## 总结

- **可贷资金市场决定真实利率**
- **利率决定资本净流出 NCO**
- **NCO 连接外汇市场，决定真实汇率**
- 预算赤字升值、贸易保护无效、资本外逃贬值。

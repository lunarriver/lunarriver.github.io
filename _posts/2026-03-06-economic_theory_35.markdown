# 一、本章核心：菲利普斯曲线（Phillips Curve）

**通胀率与失业率之间的短期取舍关系：**

- **通胀高 → 失业低**
- **通胀低 → 失业高**

# 二、短期菲利普斯曲线（向下倾斜）

### 为什么短期存在取舍？

1. 物价 / 工资有**黏性**
2. 未预期到的**总需求变化**

### 逻辑链：

总需求↑ → 物价↑（通胀↑） → 产出↑ → 失业↓

**例子：**

政府刺激经济 → 通胀从 2%→4% → 失业率从 5%→3%。

# 三、长期菲利普斯曲线（垂直）

**位置：自然失业率**

- 长期：**货币中性**
- 通胀不影响失业，失业只由劳动市场决定
- 无论通胀多高，失业都会回到**自然失业率**

**结论：**

**长期没有取舍！通胀再高，失业也不变。**

# 四、自然失业率是什么？

- 经济长期平均的失业率
- 没有周期性失业
- 长期菲利普斯曲线的**垂足**

# 五、移动菲利普斯曲线的关键：**预期通胀**

### 1. 弗里德曼 + 菲尔普斯：**预期通胀**

- 短期：**实际通胀 > 预期通胀** → 失业↓
- 长期：人们**调高预期** → 短期菲利普斯曲线**上移**
- 最终失业回到自然率

**例子：**

政府想把失业压太低 → 通胀上去 → 预期跟着涨 → 曲线往上跑 → 白忙活。

# 六、供给冲击：让曲线**整体上移**

- 油价暴涨、工资上涨、供给减少
- **短期菲利普斯曲线右移 / 上移**
- 结果：**通胀↑ + 失业↑ = 滞胀**

**例子：**

70 年代石油危机 → 成本涨 → 菲利普斯曲线变坏。

# 七、牺牲率（降低通胀的代价）

- 定义：降低 1% 通胀，要牺牲多少百分点的年 GDP

- 降低通胀：
  
  1. 紧缩政策 → 失业↑
  2. 慢慢**降低预期**
  3. 曲线下移，回到低通胀

**理性预期：**

如果大家相信央行 → 通胀可以**无痛苦下降**，牺牲率很小。

# 八、总结

短期：通胀与失业有取舍（菲利普斯曲线向下）。

长期：无取舍，失业固定在自然失业率（曲线垂直）。

预期通胀和供给冲击，会让菲利普斯曲线上下移动。

## 一、本章两个核心变量

1. **净出口 NX**（贸易余额）
2. **资本净流出 NCO**

**恒等式（最重要）：**

NX=NCO

- 每一笔贸易，必然对应一笔资本流动。
- 出口顺差 → 资本流出
- 进口逆差 → 资本流入

## 二、净出口 NX（Net Exports）

出口进口

- **贸易顺差**：出口 > 进口 → NX > 0
  
  例：中国卖很多商品给美国，赚美元。

- **贸易逆差**：进口 > 出口 → NX < 0
  
  例：美国买很多中国商品，花美元。

- **贸易平衡**：NX = 0

影响 NX 的因素：

- 国内外收入
- 相对价格
- 汇率
- 贸易政策

## 三、资本净流出 NCO（Net Capital Outflow）

本国居民买外国资产外国人买本国资产

两种形式：

1. **对外直接投资**：中国企业去美国开工厂
2. **对外证券投资**：中国人买美国国债
- NCO > 0：资本**流出**本国
- NCO < 0：资本**流入**本国

## 四、储蓄、投资、NCO 的关系（必背）

Y=C+I+G+NX

S=Y−C−G

所以：

S=I+NCO

含义：

- 一国储蓄 = 国内投资 + 资本净流出
- 储蓄多，要么国内多投资，要么借给外国。

例子：

- 中国储蓄高 → 一部分国内建厂，一部分买美债 → NCO 为正
- 美国储蓄低 → 要靠外国资本来投资 → NCO 为负

## 五、名义汇率

- 定义：**一国货币兑换另一国货币的比率**

- 例：1 美元 = 7.2 人民币

- **升值**：能换更多外币

- **贬值**：能换更少外币

## 六、真实汇率（关键）

真实汇率名义汇率国内价格国外价格

含义：

- 衡量**一国商品能换多少外国商品**
- 决定净出口贵不贵、好不好卖

例子：

- 中国商品变便宜 / 人民币贬值
  
  → 中国商品在外国更便宜
  
  → 出口变多 → NX↑

## 七、购买力平价 PPP（简单理论）

逻辑：**同样的商品，在全世界应该卖同样价格**。

- 若中国卖得便宜 → 大家来中国买 → 中国价格涨 / 人民币升值
- 最终：**1 单位本币 → 在各国购买力相同**

例子：巨无霸指数

- 美国巨无霸 $5

- 中国巨无霸 ￥20
  
  → 理论汇率：1 美元 = 4 人民币

局限：

- 很多商品**不可贸易**（理发、房子）

- 各国偏好、成本不同
  
  → 汇率**不会完全等于购买力平价**

## 总结

开放经济中，**净出口 = 资本净流出**，

储蓄 = 国内投资 + 资本净流出；

**名义汇率是钱的价格，真实汇率是商品的价格**，

购买力平价解释汇率长期趋势，但短期会偏离。

# 一、本章核心问题

1. 如何**衡量失业**？

2. 失业分哪几类？

3. 为什么总会有**自然失业**？

4. 哪些因素导致**工资高于均衡水平**，从而造成失业？

# 二、关键概念

## 1. 三个关键指标

**劳动力（Labor force）**
  
= 就业者 + 失业者
  
（有工作 + 想工作、在找工作的人）

**失业率**
  
= 失业人数 / 劳动力 × 100%

**劳动参与率**
  
= 劳动力 / 成年人口 × 100%

**示例**

某国：就业 1.8 亿，失业 0.2 亿，成年人口 2.5 亿

劳动力 = 2 亿

失业率 = 0.2/2 = **10%**

劳动参与率 = 2/2.5 = **80%**

# 三、失业的类型

## 1. 摩擦性失业

原因：**工人找工作需要时间**

特点：**不可避免、短期、正常**

例子：毕业生找工作、辞职换工作、季节性临时工结束

## 2. 结构性失业

原因：**劳动力供给 ≠ 需求**，工资**高于均衡水平**

特点：**长期**，技能不匹配、岗位消失

例子：煤矿工人因行业衰退失业，不会新技能，长期找不到工作

## 3. 周期性失业

原因：**经济衰退、总需求不足**

例子：金融危机时工厂裁员、大量失业

## 4. 自然失业率

= 摩擦性失业 + 结构性失业

**经济正常时也消除不掉的最低失业率**

# 四、造成**结构性失业**的三大原因

这些都会让**工资高于均衡水平** → 劳动力供给 > 需求 → 失业

## 1. 最低工资法

工资被定在均衡之上，低技能、年轻人最容易失业
  
**例子**：最低工资提高，餐馆少雇服务员。

## 2. 工会

工会谈判把工资抬高，工会内工人工资高，**工会外失业变多**

## 3. 效率工资

企业**主动**把工资定得很高，目的：提高生产率、减少离职、吸引更好工人

例子：华为、特斯拉给高薪，愿意招更少但更优秀的人，其他人就失业了

# 五、公共政策与失业

## 1. 失业保险

好处：保障生活、减少痛苦

坏处：**降低找工作动力，提高摩擦性失业**
  
**例子**：失业能领钱，有些人不急着找工作。

## 2. 就业培训、公共服务

减少**结构性失业**

# 六、总结

1. **劳动力 = 就业 + 失业**。

2. 失业率 = 失业人数 / **劳动力**。

3. 失业分三类：**摩擦性、结构性、周期性**。

4. **自然失业率 = 摩擦 + 结构**。

5. 最低工资、工会、效率工资 → **工资过高 → 结构性失业**。

6. 失业保险**增加摩擦性失业**。

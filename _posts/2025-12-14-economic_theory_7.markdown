本章旨在回答一个关键问题：**自由市场是否能有效配置资源，使社会总福利最大化？**


### 一、核心目标

本章通过引入两个关键概念——**消费者剩余**和**生产者剩余**，来衡量市场交易给买卖双方带来的福利，并在此基础上定义**市场效率**。

### 二、重要概念与解释

#### 1. **消费者剩余（Consumer Surplus）**

**定义**：买者愿意为某商品支付的最高价格（即“支付意愿”）减去其实际支付的价格。

**本质**：衡量消费者从交易中获得的“额外收益”或“心理盈余”。

**公式**：  

```
消费者剩余 = 支付意愿 - 实际价格
```

假设四人对一张音乐专辑的支付意愿分别为：

- Taylor：100元

- Carrie：80元

- Johanna：70元

- Grace：50元

若市场只有一张专辑，卖家通过拍卖最终以80元卖给Taylor。

- Taylor的消费者剩余 = 100 − 80 = **20元**

- 其他人未购买，剩余为0

- **市场总消费者剩余 = 20元**

若有两张专辑，均衡价为70元（第二高出价者可接受），则：

- Taylor剩余 = 100 − 70 = 30元

- Carrie剩余 = 80 − 70 = 10元

- **总消费者剩余 = 40元** → **更多交易提升总福利**

图形上，消费者剩余是**需求曲线以下、价格线以上**的面积。

#### 2. **生产者剩余（Producer Surplus）**

**定义**：卖者实际收到的价格减去其愿意接受的最低售价（即“成本”或“销售意愿”）。

**本质**：衡量生产者从交易中获得的“利润盈余”。

**公式**：  

```
生产者剩余 = 市场价格 - 生产成本
```

四位油漆工的成本分别为：

- Grandma：500元

- Georgia：600元

- David：800元

- Mary：900元

若市场价为800元，则：

- Grandma剩余 = 800 − 500 = 300元

- Georgia剩余 = 800 − 600 = 200元

- David刚好保本（剩余=0）

- Mary不接单（成本 > 价格）

- **总生产者剩余 = 500元**

图形上，生产者剩余是**供给曲线以上、价格线以下**的面积。

#### 3. **市场效率（Market Efficiency）**

**定义**：当资源配置使**总剩余（消费者剩余 + 生产者剩余）最大化**时，市场是有效率的。

**关键结论**：在**完全竞争市场**中，**均衡价格和均衡数量**自动实现总剩余最大化。

- 商品分配给**支付意愿最高的买者**；

- 商品由**成本最低的卖者提供**；

- 没有“未实现的交易收益”（即没有帕累托改进空间）。

效率判断标准：

- 若产量 **低于均衡量** → 存在“被放弃的有利交易”（买者愿付 > 卖者成本），增加产量可提升总剩余。

- 若产量 **高于均衡量** → 最后几单位商品的买者评价 < 卖者成本，减少产量反而增加总剩余。

因此，**市场均衡 = 效率配置**。

### 三、政策启示

**价格管制（如限价）会降低效率**：导致短缺或过剩，减少总剩余（如前几章讨论的排队、黑市）。

**税收、补贴等干预**会扭曲价格信号，造成“无谓损失”（Deadweight Loss）——这将在第8章深入分析。
 
**“看不见的手”**：个体追求自身利益（消费者想省钱，生产者想赚钱），却无意中促成社会福利最大化。

### 四、总结

```
总剩余 = 消费者剩余 + 生产者剩余 = （买者总支付意愿）−（卖者总成本）
```

在需求、供给曲线的均衡点处，总剩余最大 → 市场有效率

### 五、现实意义

- 解释为何自由市场通常优于中央计划；

- 为评估政府政策（如最低工资、关税、补贴）提供福利分析框架；

- 强调：**效率 ≠ 公平**——市场可能高效但结果不平等，这涉及规范分析（第8–9章及后续政策讨论）。

**一句话概括**：自由竞争市场通过价格机制，将商品匹配给最想买的人和最能高效生产的人，从而在无需外部干预的情况下，实现社会总福利的最大化。
